¿Qué aplicaciones utilizaremos en nuestra asignatura?

Access, programa de Word

Microsoft Access es un sistema de gestión de bases de datos desarrollado por Microsoft que permite almacenar, organizar y manejar grandes cantidades de información de manera estructurada. Combina una interfaz gráfica fácil de usar con potentes herramientas para la creación y administración de bases de datos relacionales.


Se utiliza comúnmente en empresas, instituciones educativas y proyectos personales para gestionar datos de clientes, inventarios, ventas, recursos humanos, entre otros. Su integración con otras aplicaciones de Microsoft, como Excel o Word, facilita el intercambio y análisis de información.


Con Access, podemos crear tablas para almacenar datos, formularios para ingresar y visualizar información de manera más amigable, consultas para buscar y filtrar registros específicos, e informes para presentar resultados de forma profesional. También es posible automatizar tareas mediante macros o código VBA (Visual Basic for Applications), lo que aumenta su funcionalidad.

En resumen, Access es una herramienta muy útil para diseñar y gestionar bases de datos sin necesidad de tener amplios conocimientos de programación.

SketchUp

SketchUp es un programa de modelado 3D intuitivo y fácil de usar, diseñado para crear, visualizar y modificar modelos tridimensionales de manera rápida. Se utiliza ampliamente en arquitectura, diseño de interiores, ingeniería, construcción y urbanismo, ya que permite representar espacios, objetos y estructuras con precisión. Su interfaz amigable y sus herramientas de dibujo permiten tanto a principiantes como a profesionales desarrollar proyectos desde bocetos conceptuales hasta modelos detallados, además de integrarse con otros programas y ofrecer una amplia biblioteca de componentes en línea.

Visual Studio Code
Visual Studio Code (VS Code) es un editor de código ligero pero potente desarrollado por Microsoft, reconocido por su velocidad, su interfaz intuitiva y su amplia compatibilidad con distintos lenguajes de programación. Cuenta con funciones como autocompletado inteligente, depuración integrada, terminal incorporado y miles de extensiones que permiten personalizar el entorno según las necesidades del usuario.

Además, su tecnología basada en Electron le permite funcionar en Windows, macOS y Linux, convirtiéndolo en una herramienta versátil tanto para desarrolladores profesionales como para estudiantes. Su comunidad activa asegura constantes mejoras y una enorme cantidad de recursos disponibles.
