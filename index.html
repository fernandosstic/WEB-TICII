Utilidades tecnológicas

Digitalización del entorno personal de aprendizaje

Diseño y desarrollo de algoritmos y programas usando lenguajes de programación, estructuras de control y depuración de errores.
Introducción a la inteligencia artificial y machine learning para crear aplicaciones que reconozcan textos, imágenes, sonidos y datos.

El bloque Digitalización del entorno personal de aprendizaje aborda el uso de herramientas digitales para organizar, crear y gestionar información en entornos tecnológicos. Incluye el diseño y manejo de bases de datos relacionales y no relacionales, prestando atención a la integridad y seguridad de los datos, así como su relación con el diseño web y la consulta eficiente de información. También se trabaja la maquetación avanzada mediante software especializado, empleando plantillas, texturas y recursos visuales que refuercen la eficacia comunicativa de los proyectos digitales. Asimismo, este bloque introduce las tecnologías de realidad virtual, aumentada y mixta, explorando su hardware, software y aplicaciones prácticas en contextos educativos y creativos. Se fomenta el uso de estas herramientas como medios para enriquecer la experiencia de aprendizaje, desarrollar la creatividad y mejorar la interacción entre el usuario y los entornos digitales.

Programación


Diseño de algoritmos y uso de lenguajes de programación para resolver problemas mediante clases, objetos, funciones y estructuras de control.

Proyecto y difusión de contenidos

Creación y publicación avanzada de sitios web con HTML, CSS y JavaScript, integrando contenido multimedia, widgets y geolocalización.

El bloque Proyecto TIC: Publicación y difusión de contenidos se centra en la creación, diseño y publicación de sitios web y materiales digitales interactivos. Los estudiantes aprenden a usar lenguajes como HTML, CSS y JavaScript para desarrollar páginas web avanzadas, incorporando elementos multimedia, widgets e interactividad que mejoren la experiencia del usuario. También se trabaja el alojamiento en servidores remotos mediante sistemas de transferencia de archivos y la integración multiplataforma a través de redes sociales o sistemas de microblogging. Además, se fomenta el trabajo colaborativo en entornos de Cloud Computing, permitiendo la edición multiusuario, la revisión y el control de cambios en proyectos compartidos. Se aborda igualmente el uso de interfaces de programación (API) para la inserción de geolocalización y otros servicios dinámicos, con el objetivo de difundir contenidos digitales de forma eficaz, accesible y comunicativa.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación esencial para el desarrollo web moderno. Fue creado para dotar a las páginas web de interactividad, permitiendo que los sitios reaccionen a acciones del usuario, manipulen elementos en pantalla y actualicen información sin necesidad de recargar la página completa. Gracias a su ejecución directa en el navegador, se ha convertido en una tecnología presente prácticamente en todos los sitios web actuales.

Entre sus utilidades más comunes se encuentran la validación de formularios, la creación de animaciones, la manipulación del DOM, la actualización dinámica de contenido y la comunicación con servidores mediante APIs. También es clave para aplicaciones modernas como chats en tiempo real, mapas interactivos, juegos en línea y dashboards actualizados constantemente.

Con el tiempo, JavaScript evolucionó más allá del navegador gracias a entornos como Node.js, que permiten ejecutar código JavaScript en el servidor. Esto abrió la puerta al desarrollo full stack solo con un lenguaje. Además, su ecosistema está impulsado por miles de bibliotecas y frameworks como React, Angular o Vue, que facilitan la creación de interfaces avanzadas, aplicaciones móviles, aplicaciones de escritorio e incluso videojuegos.

Hoy en día, JavaScript es uno de los lenguajes más demandados y versátiles, ideal tanto para principiantes como para profesionales, y un pilar fundamental en el mundo del desarrollo web.
